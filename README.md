## Skin Cancer Image Classification

This project investigates the application of machine learning and deep learning techniques for the classification of skin cancer images. Early and accurate detection of skin cancer is critical, as timely diagnosis can significantly improve treatment outcomes.

The objective of the project was to compare the performance and efficiency of different image classification approaches using both grayscale and colour images. The models evaluated included Convolutional Neural Networks (CNN), Support Vector Machines (SVM), SVM combined with Linear Discriminant Analysis (LDA), and Random Forest combined with LDA.

Models were trained using 28x28 pixel images and assessed based on classification accuracy, computational time, and the relative advantages and limitations of each approach.

Key findings:
- The SVM model trained on grayscale images achieved the highest classification accuracy (98%), although with the highest computational cost.
- The SVM + LDA model achieved comparable performance (97% accuracy) with reduced computational time.
- The Random Forest + LDA model provided the most computationally efficient solution, achieving approximately 92% accuracy with the shortest training time.
- CNN models achieved strong performance on colour images, reaching approximately 95% accuracy.

The project demonstrates the trade-off between model accuracy, computational efficiency, and model complexity when applying machine learning techniques to image classification problems.
